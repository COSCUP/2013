钻石级赞助商

Amazon Web Services(AWS)

亞馬遜雲端運算(Amazon Web Service) 是一個高度可靠(Highly Reliable)，高擴充性(Highly Scalable)， 低成本(Low Cost)的基礎設施平台。 AWS在全球各地的客戶分布超過190個國家和地區，擁有數以千計的跨國公司，政府機構以及創業團隊。自從2006年推出以來，AWS利用建立在自己後端技術平台上的技術向廣大企業提供網絡服務。

技術創新一直是公司的核心文化，驅動著亞馬遜各部門的成長。經過十多年建設和運行運行巨大規模、高度可擴展的電子商務平台(Amazon.com) 的經驗，公司總結出了一套成功經營大規模技術基礎設施和數據中心的核心競爭力，併步入了一個更廣泛的新領域 -- 為開發人員和企業提供網路服務的平台。

如今，AWS平台包括30多個不同的服務，包括彈性計算(Elastic Compute Cloud)，存儲服務(Simple Storage Service)，數據庫服務(Relational Database Service)。 AWS的客戶可以選擇最適合其業務的發展平台或解決方案，進而把焦點放在企業創新，而不是花時間與金錢在維運硬體設備上。AWS的客戶包括高速成長的創業公司，例如Spotify，Dropbox和Pinterest，也包括成熟的大型公司，例如三星，殼牌，洲際酒店集團。當然也不乏台灣企業與新創團隊，例如TrendMico, D-Link, Pixnet, Cubie Messenger, iCook… 等。

AWS 目前在全球擁有多個數據中心（美國，巴西，歐洲，日本，新加坡和澳大利亞)，您可以選擇最適合貴公司業務市場的數據中心佈署服務。建議您可以透過 http://aws.amazon.com/free/ 註冊一個AWS帳號，並開始進行測試。

若您有任何AWS的相關問題，且希望跟AWS台灣團隊聯繫，請聯絡Locarno Pan locarnop@amazon.com 或是 James Tien jtien@amazon.com 。

RS在亚太地区足迹跨越10个市场，17个当地销售办事处及7个库存量充裕的仓库。仓库分别位于奥克兰、香港、上海、新加坡、悉尼及横滨。作为亚太地区的最大经销商，我们服务超过20万工程师。

黄金级赞助

KKBOX

也許你早已認識KKBOX， 也許你從沒聽過這個名字， 有人說我們是一家科技公司， 也有人說我們是數位音樂的代名詞， 這些解釋，以上皆是。 但我們喜歡這樣形容自己： “一個不受時空限制，讓人們輕鬆享受聽音樂並與同好分享樂趣的平台。”

我們覺得能和朋友、音樂人一起分享喜歡的音樂，豐富彼此的社群生活，是一件很酷的事， 因此我們每天都在思考著， 如何讓人們輕鬆自由地透過音樂，與自己的朋友和喜歡的音樂人更靠近。

KKBOX不是音樂家，也不是明星偶像， 把網路技術玩到極致才是我們的強項。 把音樂數位化不難， 但是把音樂變得像乾淨的自來水一樣， 讓人們無論在任何系統平台，用什麼上網工具，隨時、隨地都能聽到豐富而高品質的音樂， 這就有一點門檻了。

人們想要隨時隨地享受自己喜歡的音樂也不難， 但讓大家和好朋友、名歌手一起邊聽音樂邊聊天， 這門檻又更高了。 更不要說KKBOX擁有上千萬的音樂資料庫， 從華語、英語、日語到韓語， 從流行、古典、爵士、搖滾、嘻哈到電音， 從周杰倫到棉花糖，從Lady Gaga到火星人， 只要是好聽的音樂，KKBOX絕不讓它們缺席。

你問KKBOX還有哪些強項？ 動態歌詞方便你偷偷練唱，到KTV當歌王， 放心播方便店家商場公開播放，吸引顧客上門捧場， 音樂商店方便讓習慣擁有音樂的人下載單首收藏。

簡單說，把聽音樂變得好方便、好Easy是KKBOX的最強項！ 你還在等什麼？一起聽音樂吧！KKBOX

白银级赞助

祐生研究基金會

祐生研究基金會自1978年12月30日設立登記以來，以對外一般碩士論文獎助，對內知識整理做為開端，經歷長年紮基的過程，在1987年起大力推動建築與環境永續發展相關的長期研究。在本會探究人類居住環境與永續發展的過程中發覺，近年來資訊科技與數位化的發展對於人類生活模式帶來了舉足輕重的影響，對於此領域之相關知識分享與整合，未來人才培育等等，是我們必需面對的重要課題之一。 因此，本會過去之碩士論文獎助主題除了與建築、環境、生態、健康相關議題以外，亦在2000至2008年間獎助「資訊社會與文化」類型之相關研究主題。而資訊科技與網路的發展為人類精神創發帶來新趨勢，所以在本會經過獎助方法與推廣模式的調整之後，目前首先針對國內獨立遊戲開發相關社群，推動小規模的知識與資訊分享聚會，將參訪國際遊戲開發論壇之研習心得與國內的獨立開發者交流，期能幫助國內相關社群之發展。 就其他面向的知識整合來說，過去本會曾發展出知識庫、索引書編寫、養成各科門知識執守者等知識整合應用方式，作為進行主題研究及各項開展會議時消除知識盲點、精進研討結論的利器。目前定期舉辦聯誼會及讀書會，強化祐生成員的知識密度，以面對異質多變的未來。

OSSF 自由軟體鑄造場

  3. 舉辦各類自由軟體活動及工作坊

成立於1999年11月的Linux Pilot Ltd.，是一間以Linux及開放源碼軟體知識為本的公司，其產品及服務包括網站新聞資訊、刊物出版及培訓課程，務求以全方位方式滿足各界的學習需求。 而自2001年出版的Linuxpilot月刊，更是現時亞洲市場唯一的中文Linux期刊，至今已出版超過120期，廣為亞洲中文Linux用戶所愛戴。
